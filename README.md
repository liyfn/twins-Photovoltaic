# 数字孪生场景

目前功能主要分为三大块，分别为海上风电场景、光伏场景以及科技风场景，其他还有一些公共的功能。

海上风电，目前实现的功能有风机的旋转动画、零部件爆炸图动画、零部件点击事件、底部波纹动画、不同按钮点击之间的过渡动画等。后续要添加的功能有：可手动增删风机数量、修改风机的位置、旋转角度、转速、以及名称等。

光伏，目前实现的功能有光伏方阵中逆变器的点击事件、可显示不同的工况(烟雾、火警)、无人机巡查以及无人机第一人称视角、无人机路径、不同按钮点击之间的过渡动画等。后续要添加的功能有：方阵的名称修改、逆变器扫光动画等。

科技风，目前正在准备中，难点是目前的建模达不到想要的效果。

公共功能，包括入场动画、巡检、修改巡检半径等；天气系统、时间系统；全屏等。后续可添加的功能有：孪生场景背景修改等。
